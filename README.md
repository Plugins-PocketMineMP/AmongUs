### Download/Installation

- Check Releases for **.phar** download
- Put the plugin in ``/plugins`` Folder
- and restart server and edit config.yml

### Default Configuration

``max_games: 3``

``world_name: amongus``

### Commands

- ``/au`` - Opens the Main UI for the plugin.
- ``/aum`` - Game setup UI.
